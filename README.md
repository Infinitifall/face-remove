# Face Remove

Disables rendering on hidden cubes faces for maps in Krunker.io. A significant improvement over the Krunker Editor's face optimizer.

- Handles transparent textures and opacity
- Handles rotations
- Handles faces being blocked by multiple cubes together
- Blazing fast


## Use

Paste your map export in [data/input.json](data/input.json)

```bash
make
python3 run.py
```
